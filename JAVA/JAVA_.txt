///// JAVA 

1. 자바란? 단순 프로그램이 아니라, 실행환경 개발 도구, 라이브러리를 제공.
개발에 필요한 환경을 만들어준다.
프로그램에서 라이브러리는 프로그램을 만드는데 자주 쓰이는 것들을 남들이 미리 만들어서 제공하는 것
실행환경(JRE) + 개발도구(JDK) + 라이브러리(API) => 쉽고 빠르게 어플리케이션 개발이 가능하다. 

2. 자바는 어디에 쓰이나요?
PC 애플리케이션, 이클립스, 웹 애플리케이션, Spring Framework, 모바일 애플리케이션, 안드로이드
, 빅데이터, 하둡(Hadoop), 게임(마인크래프트), 과학, 소형기기 등

3. 자바의 특징
자바는 기존의 프로그래밍 언어에 객체지향 개념을 추가한 것이다.( ex. C++, JAVA, Python, js )
자동 메모리 관리( 가비지컬렉터(gc)로 자동으로 메모리를 관리하여 프로그램 작성이 매우 편리해졌다 )
멀티쓰레드 지원( 하나의 프로그램에서 동시에 여러 작업을 할 수 있다 )
풍부한 라이브러리로 쉽게 개발 가능하다.
운영체제에 독립적( 일반적으로는 개발 시에 특정 운영체제에서만 실행 가능하다. 따라서 소스를 수정해줘야 하는데,
자바는 어느 os에서도 사용할 수 있다 )

4. 자바 가상 머신(JVM)
자바 프로그램이 실행되는 가상 컴퓨터(VM). 한번 작성하면 어디서든 실행할 수 있다( Write once, run anywhere )
여러 운영체제 별로 VM이 있어서 설치만 하면 우리가 만든 프로그램을 어디서든 쓸 수 있다.

5. JAVA API 문서의 설치
JAVVA로 프로그램을 만드는데 필요한 주요 기능을 미리 만들어서 제공
